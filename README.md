# Baseball League Manager

A simple static website template for managing a baseball league, built with HTML, CSS, and JavaScript. This template includes sections for the league schedule, team listings, and standings.

## Features

- **Schedule**: Displays upcoming and past games with dates, teams, scores, and status.
- **Teams**: Showcases league teams with logos and descriptions.
- **Standings**: A table ranking teams by wins, losses, and win percentage.

## Getting Started

1. Clone or download this repository.
2. Open `index.html` in your browser to view the site.
3. To host on GitHub Pages:
   - Push this code to a GitHub repository.
   - Go to the repository settings.
   - Scroll down to "Pages" and select "Deploy from a branch".
   - Choose the main branch and save.

## Customization

- **Data**: Edit the sample data in `script.js` to match your baseball league's information.
- **Styling**: Modify `styles.css` to change the appearance.
- **Layout**: Update `index.html` for structural changes.

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Bootstrap 5
